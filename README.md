# 📊 Sales Performance Analysis Using Python

## Horizon TechX – Artificial Intelligence & Data Science Internship Project

### Project Overview

This project focuses on Exploratory Data Analysis (EDA) of a sales dataset using Python. The objective is to clean, analyze, and visualize sales data to uncover meaningful business insights and support data-driven decision-making.
The analysis includes data preprocessing, statistical exploration, trend identification, profitability analysis, and visualization of sales performance across products, categories, and regions.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze sales and profit trends
* Identify top-performing products
* Compare category-wise and region-wise performance
* Generate actionable business insights
* Create professional visualizations for decision-making

## 📊 Dataset Information

The dataset contains sales transaction records with the following attributes:

| Column     | Description             |
| ---------- | ----------------------- |
| Order_ID   | Unique order identifier |
| Order_Date | Date of order           |
| Product    | Product name            |
| Category   | Product category        |
| Quantity   | Units sold              |
| Sales      | Revenue generated       |
| Profit     | Profit earned           |
| Region     | Sales region            |

## 💡 Key Insights

1. Technology products generated the highest revenue.
2. Laptops and Smartphones were the top-performing products.
3. Revenue was distributed across multiple regions.
4. Sales and profit showed a strong positive relationship.
5. High-value transactions contributed significantly to total revenue.
6. Product category influenced overall sales performance.
7. Regional demand varied across markets.
8. Seasonal trends affected customer purchasing behavior.
9. Higher quantities sold increased profitability.
10. Data-driven decision-making can improve business growth.

## 🚀 Business Recommendations

* Focus marketing efforts on high-performing products.
